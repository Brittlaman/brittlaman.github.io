Cpu

Dit is de CPU pagina van mijn site. CPU staat voor central processing unit. Dit wordt ook wel de processor van een computer genoemt. De processor zorgt voor de gegevensbewerking van de computer. De processor en het interne geheugen, de RAM (random acces memory) vormen samen de kernapparatuur van een apparaat. Je kunt de cpu splitsen in twee onderdelen, namelijk het rekenorgaan (alu) en het besturingsorgaan(control unit). Het besturingsorgaan let op de intructies die uitgevoerd moeten worden en het rekenorgaan voert ze uit. De processor werkt in 4 stappen. Stap 1; Ophalen van gegevens uit het interne geheugen. Stap 2; Uitwerken van de instructies door het rekenorgaan. Stap 3; Opslaan van resultaten. Stap 4; Uitvoer van de gegevens.

De snelheid van een processor

De snelheid van een processor wordt vaak weergegeven door middel van verwerkingssnelheid. Deze hangt af van de kloksnelheid en van de hoeveelheid processors die een computer heeft. Er zijn mensen bezig om een processor sneller te maken. Een manier waarop dit bereikt kan worden is de aanwezigheid van het cachegeheugen. Het cachegeheugen is een extra snel werkgeheugen, maar door de snelheid is dit geheugen ook erg duur. Om ervoor te zorgen dat de instructies sneller uitgevoerd worden kan gebruik worden gemaakt van pipelining. Deze techniek splitst de instructie is deelinstructies. De processor kan vervolgens deze instructies parralell uitvoeren, waardoor de efficiente sterk toeneemt. De pipeline stages noem je ook wel de Fetch decode execute cycle.
