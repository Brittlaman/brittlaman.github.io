CPU

Dit is de CPU pagina van mijn site. CPU staat voor central processing unit. Dit wordt ook wel de processor van een computer genoemt. De processor zorgt voor de gegevensbewerking van de computer. Je kunt de cpu splitsen in twee onderdelen, namelijk het rekenorgaan en het besturingsorgaan.
