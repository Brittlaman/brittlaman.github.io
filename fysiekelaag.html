Fysieke laag

Dit is de fysieke laag pagina van mijn website. Op deze pagina leer je over het moederbord, de bus en de randapparatuur. De cpu hoort ook bij de fysieke laag, maar daarover vind je op mijn cpu pagina meer.

Hardware-interfaces

De fysieke laag bestaat uit alle hardware elementen van een apparaat, zoals bijvoorbeeld hardware-interfaces. Dit zijn verbindingen tussen fysieke onderdelen. Vaak worden deze verbindingen mogelijk gemaakt door een printplaat, bijvoorbeeld een chip. Een voorbeeld van een hardware-interface zijn connectoren. Deze zijn aangesloten aan het moederbord van een apparaat. Het moederbord zorgt ervoor dat alle onderdelen van een computer met elkaar kunnen communiceren.

Het geheugen

Het geheugen is een van de belangrijkste onderdelen van een computer, het zorgt ervoor dat de computer zo snel is. Het geheugen van een computer bestaat uit RAM en ROM geheugen. RAM staat voor Random Acces Memory. Dit is een tijdelijk geheugen om programma's en data's op te slaan. Het is een tijdelijk geheugen omdat de geheugencellen in de chips van de computer alleen de inhoud kunnen vasthouden zolang er spanning opstaat. Een ander onderdeel van het geheugen is het ROM geheugen. ROM betekent Read Only Memory. Dit geheugen behoudt ook de informatie als de spanning wegvalt. ROM-chips worden daarom ook gebruikt voor het BIOS systeem van een computer. BIOS staat voor Basic Input/ Output System. Als je een computer opstart, dan voert het BIOS systeem een aantal testen uit en wordt het besturingssysteem geladen. Het werkgeheugen in een computer slaat tijdelijk data en programma's op. Als je permanente opslag wil, dan heb je verschillende opties, namelijk magnetisch, optisch of elektronische opslag.

Elektronische opslag

Flashgeheugen (elektronische opslag) is een manier om programma's en data voor langere tijd te behouden. De naam Flashgeheugen komt omdat de EEPROM-chips heel snel weer gewist of weer beschreven kunnnen worden. Flashgeheugen wordt bijvoorbeeld gebruikt in USB-sticks. Een voordeel hiervan is dat je alle informatie op een computer makkelijk mee kan nemen en overal kan gebruiken. Een nadeel van flashgeheugen is dat de bestandsoverdracht een stuk trager verloopt.
