Toepassingslaag

Dit is de toepassingslaag pagina van mijn website. De toepassingslaag is de eerste laag van het drie lagen model. De toepassingslaag is software. Op je telefoon zijn dit je apps, zoals instagram en somtoday. Een app kan niet zelf hardware aansturen, maar alleen contact maken met hardware via de logische laag. Er zijn software-interfaces die ervoor zorgen dat de verschillende software-onderdelen met elkaar kunnen communiceren. Deze interfaces worden API'S genoemd. API staat voor Application programming interfaces.
